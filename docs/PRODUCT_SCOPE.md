# Product Scope

## Product

AURA Studio is a deterministic AI-assisted circuit builder.

## Included

- `Circuit Studio`
  Prompt-driven circuit creation plus small manual editing.
- `Component Lab`
  Import, correct, and export reusable component packages.
- curated package library
- deterministic circuit intent and IR
- simple runtime preview for selected parts

## Excluded For Now

- host firmware
- node/locator control
- inventory/storage system
- phone app
- full analog/digital simulation
- giant vendor-wide automatic import coverage

## Success Condition

The first version wins if it can reliably create and edit common beginner circuits using a small trusted package library.
